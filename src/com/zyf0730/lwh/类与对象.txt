				担保账户：（收款账户    付款账户  ）

		类： 分类   根据什么分类？ --  相同或相似的属性、特征、行为方式、功能
			程序中： 具有相同的属性、特征、行为方式、功能的一类对象的模板
		
		对象：具体的事物  一个对象就是一个具体的事物 
			程序中： 由类创建的 - 存在于内存中 
		
	类的编写： 可以创建程序运行的对象的类  其他的是 工具类 	
		类- class  
	用户类 ：
		属性：姓名  年龄 手机号 地址 账户等级 ....
		方法：观看无广告   观看VIP视频   下载 .... 
	属性： 一个数据、一个变量能表达的	 
		格式： 数据/属性类型   属性变量名;
		类型:  基本数据类型：int  double char 
				类- 类型： String  只要是一个类就可以作为类型存在 
		String name;  
		 
 	方法： 功能或行为的描述 
	public void test(int n){ }
	格式： 访问修饰符  返回值类型  方法名（参数类型  参数变量名）{
		// 方法体
	}
		返回值：  有-int/String 	return 对应类型的数据/变量
				 空 - void 		不用 return
				 无 - 构造方法  没有返回值类型这个结构		
		参数：相当于 方法的局部变量，可以与属性重名，但是在方法中就近使用
			有：
				单个（参数类型  参数变量名）
				多个（参数类型1  参数变量名1,参数类型1  参数变量名2,参数类型2  参数变量名3）
				
			无： 空着 不写 
	类创建对象
		关键字： new 
		格式： 类名 对象变量名  = new 类名（）;
		
		对象复制了一份属性和方法存在自己内存空间中 
		1、对象都是独立的

		程序入口 -- 主方法
