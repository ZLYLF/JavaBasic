	接口： interface
		格式： public interface 接口名{      }
		
		1、不能定义变量
		2、不能定义带方法体的方法   - 抽象方法  
		
	接口可以创建对象吗？		不可以
	接口可以作为一种类型存在吗？	可以

	使用接口：行为/方法 的抽象 
		创建一个类来实现接口   - implements 
		格式： public class 类名 implements 接口{    }
			1、必须实现接口中所有的抽象方法  
			2、这个类可以直接使用接口中的常量
		 拥有了接口中的常量
		 实现了接口中所有的方法 
		 这个类的对象 可以自动转型为接口类型
