	
	界面- 登录界面 
		
		Java项目结构 :
		 	JRE - java 源代码 
		 	src - 自己写的代码
		 		- package  - 包  给代码进行分类 
		 			- 类 -   .java 后缀的代码 
		 			
		 			
		java.awt 	早期的界面开发包     保留使用元素类的使用 
		javax.swing  升级的界面开发包 	  可视化组件的使用 
		
		按照功能分类：
		输入框   复选框    单选框   下拉框   按钮   标签（文字  图片 ）窗体
 javax.swing容器： 顶级容器 -  窗体（自带  最小化 最大化 标题栏 关闭按钮 ）
				    面板		
 javax.swing组件：输入框   复选框    单选框   下拉框   按钮   标签（文字  图片 ）.... 
	java.awt元素：尺寸   颜色  字体  布局 
				布局  - 控制组件的摆放规则 
			JFrame -- 默认布局 边框布局 
				东西南北中 - 每个方位只能放置一个组件或容器 
				默认为中-  除了东西南北之外所剩下的区域 
				南北的宽不受尺寸影响 ，东西的高不受尺寸影响，是除掉南北的高剩下的高度 
			JPanel -- 默认布局 - 流式布局 
				依次从左至右摆放组件，摆不下时切换下一行摆放，默认居中
	步骤：
		1、搭建一个底层容器 -- 创建一个窗体对象  -- 设置窗体的属性 
		2、设置布局方式  
		3、创建组件对象 
			设置组件的对象的属性 
			将组件对象加载到窗体上
