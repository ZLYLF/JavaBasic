		类：具有相同属性、特征、行为方式、功能的一类事物的集合
			程序中： 具有相同属性、方法的一类对象的模板
		对象：具体的事物，一个对象就是一个具体的事物 
			程序： 由类创建的 - 存在于内存中 
				程序不运行 没有对象存在 ，只不过是一行代码

	类的编写：
		关键字： class 
		格式： public  class 类名{  }
		类名命名： 首字母大写，其后的每个单词首字母大写  -- 驼峰命名法 
		变量名，方法名：  首字母小写，其后的每个单词首字母大写
	
	属性：可以由一个数据，一个变量表达的 String name;
		格式： 属性类型  属性变量名；
		类型： 
			基本数据类型：char int double  
			类-数据类型： String  只要是一个类据可以作为一种类型
						更为高级 复杂；


	方法: 行为、功能 的描述 
		格式： public  返回值类型 方法名（参数类型 参数变量名）{  // 方法体 }

		返回值：		返回到调用这个方法的代码上 
					 返回值类型				return 返回值
				有：	 int / String 				10 / "name" - i / str
				空：   void  						不需要return
				无：	  不写	-构造方法				不需要return
				
		参数：	参数相当于方法的局部变量，可与属性变量重名，但是在这个方法中 就近使用
				有：
					单个：（参数类型 参数变量名）
					多个：（参数类型1 参数变量名1，参数类型1 参数变量名2，参数类型2 参数变量名3）
				无：
					空着
					
		
	类创建对象：
		关键字： new 
		格式： 类名    对象变量名 = new 类名（）；
		
		创建对象时发生了什么事情？
			对象复制了一份属性和方法存在自己的内存空间中
